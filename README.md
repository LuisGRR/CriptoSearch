## Tecnologias utilizadas

## Glonar proyecto

```git
git clone https://github.com/LuisGRR/CriptoSearch.git
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
